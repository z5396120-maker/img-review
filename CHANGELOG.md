# Changelog

## 0.1.0 - 2026-06-19

- Initial open-source release as Img Review.
- Added local Codex plugin manifest and `img-review` skill.
- Added browser-based image annotation canvas with box, arrow, draw, point, and Magic selection tools.
- Added brush-based Magic selection, add/subtract selection modes, transform handles, undo/redo, drag-and-drop image import, and before/after transform indicators.
- Added structured `annotations.json`, `review.md`, `ai-task.json`, and `ai-task.md` export flow.
